QUESTION 231

次のプログラムの実行した際の表示結果を答えよ。 （1）x = 3 （2）y = 4 （3）goukei = x + y （4）表示する("合計は",goukei,"です。")

答えを見る

答え合計は7です。

キーワード
  • このプログラムは変数への理解を問うものである。 前回、変数を「箱」に喩えたが、今回は「ラベル（名札）」と喩えられる。 このとき、「＝」は「ラベルを貼り付ける（関連付ける）」ことと理解できる。 1行目・2行目：3，4という値にそれぞれx，yという名前のラベルを貼り付ける。 3行目：ラベルxがついた値（3）とラベルyがついた値（4）を足しあわせたもの（7）にgoukeiという名前のラベルを貼り付ける。 4行目：「合計は（ラベルgoukeiがついた値）です。」と表示する。つまり、「合計は7です。」と表示し、プログラムは終了する。 初心者にとっては、変数を「箱」と考えるほうがわかりやすいかもしれない。 代入される値が数値や文字列の場合、「箱」と考えて十分に理解できる。 参照型変数やオブジェクト指向プログラミングを理解するには、変数を「ラベル」と考えるほうがより理解が深まる。

          QUESTION 237

          次のプログラムの実行した際の表示結果を答えよ。 （1）x = 10 （2）x = x * x （3）表示する("現在のxの値は", x, "です。")

          答えを見る

          答え現在のxの値は100です。

          キーワード
          • このプログラムは変数が自身の値を更新するものである。 1行目：変数xに値10を代入する。 2行目：最初に右辺の計算が行われる。変数xの現在の値は10なので、10×10＝100が計算される。この100を左辺の変数xに代入する。つまり、変数xの値は現在の10から右辺の計算結果である100へと更新される。 3行目：「現在のxの値は100です。」と表示して、プログラムは終了する。 2行目「x ＝ x * x」を更に詳細に解説する。 重要な点は、「＝」が「数学の等号：左辺と右辺が等しい値である」という意味でなく、「左辺に右辺を代入せよ」という指示であることである。 代入「＝」する際は、まず右辺を計算し（x * x = 10×10 = 100）、その結果を左辺（x）に代入する。 つまり、右辺の変数xの値は、この行が実行される直前までに変数xに代入されていた値であり、それを使った計算結果を左辺の変数xに代入し、変数xの値が更新されるという流れである。

            QUESTION 239

            次のプログラムの実行した際の表示結果を答えよ。 （1）x = "あいう" （2）y = "えお" （3）表示する(x + y)

            答えを見る

            答えあいうえお

            キーワード
            • 変数には数値だけでなく文字列も代入することができる。 共通テスト用プログラム表記では"ダブルクオーテーション"で囲われたかたまりを文字列として扱う。 また、「＋」は文字列同士の連結を表す。 つまり、このプログラムは以下のように理解できる。 1行目：変数xに文字列"あいう"を代入する。 2行目：変数yに文字列"えお"を代入する。 3行目：変数xと変数yを連結し、その結果を表示する。よって、「あいうえお」と表示し、プログラムは終了する。

            QUESTION 240

            次の2つのプログラムのした際の表示実行結果を答えよ。 プログラムA （1）x = 1 （2）y = 2 （3）表示する(x + y) プログラムB （1）x = "1" （2）y = "2" （3）表示する(x + y)

            答えを見る

            答えプログラムA：3 プログラムB：12

            キーワード
            • プログラムAとプログラムBでは変数に代入されるものに違いがある。 プログラムAで代入される値は数値であるが、プログラムBでは"ダブルクオーテーション"で囲われているため文字列として扱われる。 この点に注意して、プログラムを詳細に見ていく。 プログラムA 1行目：変数xに数値1を代入する。 2行目：変数yに数値2を代入する。 3行目：変数x，yの値が数値なので、「x + y」は数値同士の足し算を意味する。 つまり、1＋2を計算し、「3」と表示する。 プログラムB 1行目：変数xに文字列"1"を代入する。 2行目：変数yに文字列"2"を代入する。 3行目：変数x，yの値が文字列なので、「x + y」は文字列の連結を意味する。 つまり、文字列"1"と"2"を連結し、「12」と表示する。 数値と文字列でプラス記号の意味が変化するため、注意が必要である。
