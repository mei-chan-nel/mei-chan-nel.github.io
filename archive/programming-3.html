QUESTION 251

映画館の座席が5行3列の2次元配列Seatsで管理されている。
座席は0が空席、1が予約済みを表す。
現在のSeatsの要素の値は以下の通りになっている。
なお、この配列の添字は1から始まる。
0 1 0
1 1 0
0 1 0
0 0 1
1 0 0
次のプログラムの実行した際の表示結果を答えよ。

（1）gyou = 3
（2）retu = 2
（3）表示する( gyou, "行", retu, "列目の座席の状態は", Seats[gyou, retu], "です。")
答えを見る

答え3行2列目の座席の状態は1です。 （予約済みです）
