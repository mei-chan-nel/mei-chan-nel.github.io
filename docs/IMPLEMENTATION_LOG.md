## 2026年7月14日: 動画問題の可読性とキーワード監査

- Q001〜Q330を問題文・正答・分類・対応動画題名から1問ずつ見直し、既存値を参照せずキーワードを再構築した。

- Google検索セントラルのSEOスターターガイド、ユーザー第一のコンテンツ、AI機能、リンク、サイトマップの公式説明を基準に再監査した。判断と公開後の測定方法は `docs/SEO_IMPLEMENTATION.md` に記録した。
- トップのtitle、description、H1を高校生・受験生の検索意図に合わせ、「情報Ⅰ」「情報1」「共通テスト」を自然な文脈で明示した。
- 独自の学習コンテンツとして `study-guide.html` を追加し、5段階の学習手順、6分野、3か月・1か月・1週間の計画、つまずき、FAQ、文科省・大学入試センターの公式資料を掲載した。
- 各ページへ固有のメタ情報、Open Graph、canonical、パンくず構造化データを追加した。トップはWebSite、勉強法はArticle、一覧はCollectionPageで内容と一致させた。
- アプリ掲載1,000問では4問以上に付く公開タグ、動画330問では90キーワードをリンク化した。各一覧は複数選択でき、いずれかを含む問題をURL付きのOR検索で抽出する。

- アプリ掲載問題のタグは全1,000問で4問以上に付くものだけを公開する基準へ変更した。元データ613種類のうち公開244種類、非表示369種類となり、公開タグが0件となる10問ではタグ欄自体を表示しない。生成レポートと検証レポートに基準値と件数を記録する。
- `robots.txt`で`OAI-SearchBot`を明示的に許可した。AI検索だけを目的とする独自ファイルや非公式マークアップは追加していない。
